# Clean Phase is designed so the Goose cannot win in Phase 1

The Clean Phase window (5–10s) is intentionally too short to fully clean the Room as Bacteria tiers escalate. The Filth Gauge is designed to trend upward regardless of player skill — the Goose is always falling behind.

This is deliberate: the game is a controlled descent. As the Filth Gauge climbs toward 100%, the Goose is eventually overwhelmed and forced into Phase 2 (Goose Boss Mode). The player cannot "play well enough" to stay in Phase 1 forever. Losing Phase 1 is the intended path to winning the game.

**Considered alternative:** A generous Clean Phase window that rewards skilled play and allows the Filth Gauge to stay low. Rejected because it removes the narrative arc — the Goose's transformation into a boss should feel inevitable and earned, not optional.
